# How PolyNotice Works

PolyNotice operates as a layered system combining monitoring, analytics, and automation.

## 1. Market & Wallet Monitoring
PolyNotice continuously tracks:
- Market prices and probabilities
- Liquidity and volume
- Wallet activity (user + tracked accounts)

## 2. Alert Engine
Users define conditions such as:
- Price targets
- Percentage movement
- Multi-condition triggers

When conditions are met, alerts are triggered.

## 3. Intelligence Layer
PolyNotice analyzes data to generate insights:
- Conviction Heatmap (market activity ranking)
- Profit leader tracking
- Market opportunity detection

## 4. Automation (Planned)
Users will be able to:
- Set take-profit / stop-loss rules
- Enable non-custodial execution

All automation is user-defined and permission-based.
